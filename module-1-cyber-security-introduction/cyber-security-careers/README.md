# Careers in Cyber Security

Room completed on TryHackMe.

## Overview

Cybersecurity offers many different career paths depending on the professional's interests and technical skills.

Organizations rely on cybersecurity professionals to protect systems, detect threats and respond to incidents.

## Career Paths

Some common roles in cybersecurity include:

- Security Analyst
- Penetration Tester
- Incident Responder
- Security Engineer
- Threat Intelligence Analyst
