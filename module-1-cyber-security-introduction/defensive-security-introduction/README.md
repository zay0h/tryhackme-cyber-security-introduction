# Introduction to Defensive Security

## Overview

Defensive security focuses on protecting systems, detecting attacks and responding to security incidents.

Security professionals working in defensive roles monitor networks, analyze suspicious activity and help organizations defend against cyber threats.

## What I Learned

- The difference between offensive security and defensive security
- The role of Blue Team in cybersecurity
- The importance of monitoring and detection
- Basic incident response concepts
